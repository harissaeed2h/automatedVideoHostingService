-------------------------------- README ---------------------------------------- THIS FOLDER NEEDS TO BE HOSTED VIA ANY SERVICE (e.g: APACHE, Python Http server, etc) ------------------


-----> Run the update.sh script to update the website to contain all updated video and image files.

-> Place the images/photos inside the images directory.
	-> which will be hosted on the photos section of the website. The photos directory is only for the html,css, and js files.

-> Place Video Files into videos directory.
	-> videos page is the main page, loads up as the main webpage.

-> videosIndex.json:
	-> this file is created by the python file.
	-> this file contains the titles for the videos, their video file locations, their hashes(for dynamic web URLs), and locations to their thumbnail files generated by the python script

-> imagesIndex.json:
	-> simillar to the videosIndex.json
	-> contains only titles and file locations
